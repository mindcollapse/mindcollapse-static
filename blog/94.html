Simple eBay PHP wrapper

13 февраля 2008 г.
Vladimir Smirnov
В хо­де ра­бо­ты над од­ним круп­ным про­ек­том на­пи­сал ма­лень­кую оберт­ку для eBay shopping API. Ска­чать класс (php >= 5.1.0) мож­но по ссыл­ке.До­ку­мен­та­цию пи­сать не ви­жу смыс­ла, класс очень про­стой. На вы­хо­де по­лу­ча­ем SimpleXMLElement Object. Ре­зуль­та­ты ке­ши­ру­ют­ся в фай­лах, за­прос идет че­рез lib_curl, от­вет раз­би­ра­ет­ся simplexml. Все про­сто и удоб­но, чи­тай­те код (все­го-то 83 стро­ки) и ма­ну­ал по API ме­то­дам для до­пол­ни­тель­ной ин­фор­ма­ции.
